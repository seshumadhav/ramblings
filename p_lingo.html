Lingo

  • Locality of Reference Principle
    • "A program will access a particular set of data over a short period of time"
    • Temporal locality - If a memory location is accessed, it is likely to be accessed again soon.
    • Spatial locality - if a memory location is accessed, nearby memory locations are likely to be accessed soon.
  • Pareto Principle (80/20 rule)
    • "80% of effects come from 20% of causes"
    • A small portion of inputs often leads to the majority of outputs or results
  • Parkinson's Law
    • "Work expands so as to fill the time available for its completion"
  • Bloom Filters (source: how.dev)
    • A probabilistic data structure that uses bit vectors. Does not store actual data.
    • It can confirm absence of a Key accurately and quickly.
    • However presence cannot be confirmed.
  • Ephemeral (source: chatgpt.com)
    • Something short-lived or temporary.
    • Data or resources exist only for a brief period and can be destroyed/recreated without consequence.
  • Bespoke Solution vs Customized/Tailored Solution
    • Bespoke - Built from scratch exclusively for a specific organization or client or team
    • Customized - Started with an off-the-shelf solution and tweaked it
  • Off the shelf solution
    • A ready-made, mass-produced product that you can buy and use as-is.
    • Eg: CRM solutions like Salesforce, Google Office Suite
  • Out of the box solution
    • Works immediately upon setup or installation, with no (or minimal) configuration needed.
    • Eg: Zoom, slack
